# Money - Finance Tracker

This is a minimalist, aesthetic personal finance tracker for managing income, expenses, and savings with beautiful visualizations.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
